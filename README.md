# ⚽ Head Soccer Arcade

An action-packed 2D web arcade football game built with HTML5 Canvas, pure JavaScript, and Web Audio API. Play solo against an AI opponent, test your skills in penalty shoot-outs, or challenge a friend in local couch PvP!

🎮 **[Play Live Demo Here](https://yahhiya.github.io/football-game/)**

---

## 🔥 Features
* **Flexible Match Options:** Choose between a standard full match (with extra time and penalty tie-breakers if level) or dive straight into a dedicated Penalty Shoot-out mode.
* **Dynamic Power-Ups:** Collect random on-field power-up stars that spawn during matches, featuring speed boosts, goal-size modifiers, opponent freezes, and player scaling buffs.
* **Custom AI & Physics:** Play against a responsive single-player AI or face off locally with balanced player collision physics and high-loft shooting mechanics.
* **Retro Audio:** Custom synthesised sound effects powered directly by the Web Audio API.

---

## 🕹️ Controls

### 1-Player Mode (vs AI)
* **Move / Jump:** `W`, `A`, `S`, `D`
* **Low Kick:** `J`
* **High Kick:** `K`

### 2-Player Mode (Local PvP)
| Action | Player 1 (Left Side) | Player 2 (Right Side) |
| :--- | :--- | :--- |
| **Move / Jump** | `W`, `A`, `S`, `D` | `Arrow Keys` |
| **Low Kick** | `C` | `O` |
| **High Kick** | `V` | `P` |

* **Pause Game:** `Escape`
* **Post-Match Menu:** Press `R` to Rematch or `M` to Return to Main Menu.

---

## 🛠️ Built With
* **HTML5 Canvas** – 2D rendering engine
